    Projects

    Projects all lead to their GitHub respository, where there is more documentation and detail. I have ordered these in terms of effort in descending order.

    Bookstore Web Scraper

    A bookstore web scraper to extract book information from an online catalogue using xlml, scrape that information asynchronously using asyncio and RabbitMQ to queue work items, store information in a postgreSQL database, allow users to query data from the created database using Flask, and pull data in a way that allows for additional book information than the website provides. The project was containerized using Docker.

    Legends of Pygame

    A work-in-progress game built from the ground up in Python using the Pygame library. Incorporates graphics, collisions, classes for various entities, world maps, animations, maps that update with player position, and more.

    Multiple File Search and Execute

    The program takes in command line arguments and presents different data to the user depending on the flags the user provides. All flags can be used in conjunction with one another to produced subsequently more limited results for the data output. The data can be limited based on file size, file type, file name, and the user can execute any command on the specfically selected data.

    Bookstore Manager

    A Java Swing GUI that a bookstore manager could use to keep track of the inventory of his bookstore, all of his customers, and which customers were renting which books.

    Alter CSV in SQL and Present in Python

    Takes in CSV data in python, passes that data to MySQLWorkbench alters the data in SQL, and using Pandas, returns a table of the result as well as updating the CSV data.

    AI Image Caption Generator

    Uses the Torch and Lavis Libraries to implement image captioning and visual question answering.

    PDF Data Transfer to Excel Table

    Reads pdf tables using Tabula and transfers that data into excel using Pandas library to increase ease of access to the data. The data is then stored in a local folder. Project was containerized using Docker.
